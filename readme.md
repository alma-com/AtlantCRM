## AtlantCRM

1. Install composer packages

        composer install

1. Generate key application

        php artisan key:generate

1. Run migrate command

        php artisan migrate

1. Permission update

        php artisan permission:update

1. Create admininstrator

        php artisan user:admin {email} {password}
